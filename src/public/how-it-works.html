Experience Banking Without Internet

HPesa works on any mobile phone with USSD capability. No smartphone or data connection required.

  • Instant transactions (3-5 second settlement)
  • Ultra-low fees ($0.0001 per transaction)
  • Bank-level security with PIN protection

2. Core Features

Send HBAR

  1. Dial *123#
  2. Select "Send Money"
  3. Enter recipient's phone or Hedera ID
  4. Enter amount
  5. Confirm with PIN

Buy Crypto

  1. Dial *123#
  2. Select "Buy Crypto"
  3. Choose currency (BTC, ETH, HBAR)
  4. Enter amount in local currency
  5. Confirm with PIN

Check Balance

  1. Dial *123#
  2. Select "Check Balance"
  3. View HBAR and local currency balance
  4. Option to receive via SMS

3. Behind the Scenes

1

USSD Request

User dials *123# and selects action

2

PIN Verification

System decrypts private key with user's PIN

3

Hedera Transaction

TX signed and broadcast to Hedera network

4

Confirmation

User receives SMS with TX details
